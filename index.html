Element

Ett HTML element är en komponent och används för att rendera webbsidor som är ubbyggt i strukturen HTML. Element kan ha flera attribut och elementet består oftast av en start och slut tagg.

Exempel:

HTML

HTML är förkortning för Hyper Text Markup Language. Det används för att skapa webbsidor. HTML beskriver strukturen för en webbsida. I HTML finns det även många olika element och elementet bestämmer vad webbläsaren ska visa.

Exempel:

ID

ID är ett attribute som specifierar ett unikt id för ett element. Det unika id används sedan för att refferar till elementet i exempelvis javascript eller css. ID kan också enbart användas för att ett element och inte flera.

Exempel:

Class

Class är ett attribute som specifierar ett unikt id för ett element. Det unika id används sedan för att refferar till elementet i exempelvis javascript eller css. Skillnaden mellan ett ID och en class är att ID enbart kan användas för at t identifiera ett element, medans en class kan användas för att identifiera flera.

Meta taggen i HTML dokumentet definerar metadata om ett HTML dokument. Meta taggen hittas alltid i taggen head taggen.

Http

Http står för Hypertext Transfer Protocol och är ett kommunikationsprotokoll som används för att överföra och ladda upp webbsidor på internet.

Https

Https står för Hypertext Transfer Protocol Secure som är ett protokoll för http fast krypterat och mer säkert.

Compressed data

Compressed data är data som har omkodats på ett mer effektivt sätt för att färre enheter ska behöva användas under transfer processen. Datan blir enkelt sagt mindre komprimerad.

CDN

CDN är ett innehållsleveransnätverk och hänvisar till en distribruerad grupp av servrar som arbetar tillsammans för att ha en effektiv leverans av internetinnehåll, som exempelvis HTML-sidor, js filer, bilder och videor.
